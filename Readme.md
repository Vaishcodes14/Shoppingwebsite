# 🛍️ DRAPE – Fashion E-Commerce Website

A modern and responsive fashion e-commerce website built using HTML, CSS, and JavaScript.  
This project showcases a stylish UI with product listings and interactive elements.

---

## 🚀 Features

- 🛒 Product listing section (Dresses, Tops, Kurtas, Co-ords)
- 🎨 Modern UI with clean design
- 💻 Fully responsive layout (mobile + desktop)
- 🔍 Search bar UI
- 🧭 Category-based navigation
- 🛍️ Add to Cart button (basic functionality)
- ✨ Hover effects on product cards
- 🤖 AI chatbot UI (assistant interface)
- 📦 Cart sidebar design
- 🎯 Smooth user experience

---

## 🧑‍💻 Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla JS)
